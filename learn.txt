findOne 함수 사용법

(name : title 이라던지) 에 해당하는 값을 result에 담고, 그 result를 res.render 함수의 2번째 파라미터로 json 형태로 입력한다.

예)
( name : title은 예시이다. name이라는 키의 값이 title인 데이터를 가져오겠다는 의미이다. )
( watch.ejs 파일 내에서 test라는 이름으로 findOne으로 가져온 값을 쓸 수 있다. )

database.collection('컬렉션 이름').findOne({ name : title }, function(error, result){
    실핼 할 코드
    res.render('watch.ejs', { test : result })
})
